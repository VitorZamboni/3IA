Instalando Principais Redistribuiveis

Redistribuíveis são componentes de software que precisam ser distribuídos juntamente com um programa para que ele funcione corretamente em outro sistema ou computador. Esses componentes geralmente incluem bibliotecas de código, frameworks, drivers e outros arquivos necessários para executar o programa. Nesse artigo iremos mostrar como instalar os principais redistribuiveis no sistema operacional Windows 10.

Microsoft Visual C++

Microsoft Visual C++ Redistributable é um conjunto de bibliotecas que permitem que programas desenvolvidos em C++ sejam executados em computadores que não possuem o compilador C++ instalado

Para instalar Microsoft Visual C++ entre no site da Microsoft e instale a versão X64.

Em seguida abra o executável que foi baixado, aceite os termos e clique em instalar.

Para verificar se você já tem o Microsoft Visual C++ instalado em sua máquina, ou se a instalação ocorreu corretamente, abra o painel de controle e clique em Programas.

Após isso, vá para programas e recursos.

Procure por "Microsoft Visual C++" na lista de programas instalados

Java Runtime Environment (JRE)

Java Runtime Environment (JRE) é uma plataforma de software que permite a execução de aplicativos Java em diferentes sistemas operacionais. Para fazer baixar dessa ferramenta entre no site de download do Java e clique em "Download Java".

Após baixar o instalador, abra-o e clique em instalar.

Se ao final do processo você ver uma tela parecida o a abaixo, parabéns agora você possui Java em seu sistema e está apto a executar aplicativos Java.

Para verificar se o Java está instalado corretamente, abra o Prompt de Comando ou Power Sheel e digite o comando java -version . Se voce obter uma saída parecida com a abaixo quer dizer que o Java está instalado em seu PC.

DirectX

DirectX é uma coleção de APIs (interfaces de programação de aplicativos) usadas principalmente para jogos e aplicativos multimídia que requerem recursos gráficos avançados. O principal objetivo do DirectX é fornecer uma plataforma unificada e de alto desempenho para desenvolvedores de jogos e aplicativos multimídia. Ele oferece suporte a uma ampla gama de recursos, incluindo gráficos 2D e 3D, áudio, entrada de dispositivos, rede, entre outros.

Para baixar o DirectX na sua versao mais recente entre no site da Microsoft e clique no botão "baixar".

Após baixar o executável, abra-o, aceite os termos de uso e clique em "avançar".

Então seguirá para uma janela onde você pode optar por intalar a Barra do Bing, que refere-se a uma barra de ferramentas fornecida pela Microsoft, que inclui recursos de pesquisa e acesso rápido a serviços relacionados ao Bing, o motor de busca da Microsoft. Marque ou desmarque a caixa conforme a sua necessidade de uso e em seguida clique em avançar.

Em seguida aparacerá uma tela para iniciar a instalação do DirectX, clique em "avançar" novamente.

Depois de um tempo a instalação finalmente será concluida, basta clicar em "concluir" e você terá o DirectX instalado em seu computador.

.NET Framework

O .NET Framework é uma plataforma de desenvolvimento de software da Microsoft, que fornece um ambiente para a criação e execução de aplicativos Windows. Ele inclui um conjunto de bibliotecas e componentes necessários para executar muitos aplicativos desenvolvidos com base no .NET Framework.

O Windows 10 normalmente inclui uma versão específica do .NET Framework pré-instalada, que pode variar dependendo da versão do sistema operacional e das atualizações instaladas. Caso queira realixar o download de uma versão específica do .NET pode faze-lo acessando o site da Microsoft

Concluindo

Este artigo trouxe um tutorial de como instalar os principais redistribuiveis do mercado, esses componentes são muito necessários no nosso dia-a-dia, pois muitos programas dependem deles instalados no computador para funcionarem. Portanto, não deixe de te-los instalados e atualizados em sua máquina.
